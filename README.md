AI Telegram Memory Bot (Work in Progress)

This project is an experimental AI-powered Telegram bot designed to connect the Telegram API with another AI model’s API, featuring memory management and a personality plugin.

What It Does

Bridges Telegram messages to an AI backend.

Implements basic memory chunking to manage AI context limits.

Adds a personality plugin to customize bot responses.

Encrypts memory data before saving (incomplete feature).

Current Status

⚠️ Work in Progress / Abandoned

The bot is not complete and may have bugs or missing functionality.

Encryption and memory management features are partially implemented.

Intended as a proof of concept and learning exercise.

Why It Matters

This bot explores solving AI memory limits by chunking and encryption, plus integrating personality traits into AI responses. Even though incomplete, it demonstrates ideas that can be built on.

How to Use

Clone the repo.

Install dependencies (see pyproject.toml).

Configure your Telegram bot token and AI API keys.

Run main.py.

Note: This project is a work in progress, currently incomplete, and should not be used in production environments.
